# NOVA PIE

Futuristic pizza landing page with immersive UI animations and interactive sound effects.

## Description
NOVA PIE is a static front-end project designed with a sci-fi visual style, motion-rich interactions, and a responsive layout for desktop and mobile. It includes animated hero effects, interactive menu cards, custom pizza builder logic, and Web Audio API based UI sound effects.

## Live Link
- Production: https://pizza-seven-mocha.vercel.app

## Features
- Futuristic UI design with animated gradients, rings, particles, and reveal effects
- Interactive menu filtering and add-to-cart feedback
- Custom pizza builder with dynamic price and flavor score updates
- Optional UI sound effects (`SFX ON/OFF` toggle)
- Fully static and Vercel deployable

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Vercel (deployment)

## Project Structure
- `index.html` - Page structure and content
- `style.css` - Visual design, layout, and animations
- `script.js` - Interactivity, sound effects, and dynamic UI logic
- `vercel.json` - Deployment settings for Vercel

## Run Locally
1. Clone the repository.
2. Open `index.html` directly in a browser, or use a simple static server.

## Deploy
This project is already configured for Vercel. Pushes to `main` can be auto-deployed when connected to GitHub.
